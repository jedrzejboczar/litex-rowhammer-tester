SDRAM_DFII_CONTROL
^^^^^^^^^^^^^^^^^^

`Address: 0x00004000 + 0x0 = 0x00004000`

    Control DFI signals common to all phases

    .. wavedrom::
        :caption: SDRAM_DFII_CONTROL

        {
            "reg": [
                {"name": "sel",  "attr": '1', "bits": 1},
                {"name": "cke",  "bits": 1},
                {"name": "odt",  "bits": 1},
                {"name": "reset_n",  "bits": 1},
                {"bits": 28}
            ], "config": {"hspace": 400, "bits": 32, "lanes": 4 }, "options": {"hspace": 400, "bits": 32, "lanes": 4}
        }


+-------+---------+-----------------------------------------+
| Field | Name    | Description                             |
+=======+=========+=========================================+
| [0]   | SEL     |                                         |
|       |         |                                         |
|       |         | +-------+-----------------------------+ |
|       |         | | Value | Description                 | |
|       |         | +=======+=============================+ |
|       |         | | 0     | Software (CPU) control.     | |
|       |         | +-------+-----------------------------+ |
|       |         | | 1     | Hardware control (default). | |
|       |         | +-------+-----------------------------+ |
+-------+---------+-----------------------------------------+
| [1]   | CKE     | DFI clock enable bus                    |
+-------+---------+-----------------------------------------+
| [2]   | ODT     | DFI on-die termination bus              |
+-------+---------+-----------------------------------------+
| [3]   | RESET_N | DFI clock reset bus                     |
+-------+---------+-----------------------------------------+
